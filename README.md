# MicroBudget Planner

A **simple, local-first budgeting web app** designed for people who find traditional budgeting tools too complicated.

**Features:**  
- Add income and expenses with categories  
- View all transactions in a clean, animated list  
- Search and filter by category, date, amount, or type  
- Visualize spending with animated charts  
- Set weekly reminders  
- Export and import data as JSON  
- Fully offline and stores all data locally in the browser  

**Tech Stack:**  
- HTML, CSS, Vanilla JavaScript  
- LocalStorage / IndexedDB  
- Canvas/SVG for charts  
- Responsive design with smooth animations  

**Goal:**  
Make personal finance simple, intuitive, and fun, without requiring accounts, APIs, or servers.
